“Deep learning-based brain tumor segmentation and growth prediction using a hybrid framework combining exponential growth modeling with a neural network-based correction model.”

Overview
This project focuses on segmenting brain tumors from MRI scans using deep learning and predicting tumor growth using a hybrid approach combining an exponential mathematical model with a correction model.

Objectives
Segment tumor regions from MRI images
Estimate tumor volume
Model tumor growth over time
Improve prediction using a correction model

Approach
• Develop a deep learning model (U-Net) for accurate brain tumor segmentation from MRI images
• Extract tumor regions and compute tumor size from segmentation outputs
• Implement a baseline model for tumor growth prediction
• Design a neural network-based correction model to improve growth predictions
• Integrate segmentation and growth prediction into a unified automated pipeline
• Enable faster and more consistent clinical decision support

Tech Stack
Python
PyTorch
OpenCV
NumPy, Pandas

How to Run
Clone the repository
Install dependencies using requirements.txt
Run the main script
